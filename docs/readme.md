숫자 스무고개

행위

1. 컴퓨터가 숫자 난수를 생성한다.
    - Math.random()을 사용
    - 난수는 1~100 사이의 숫자여야 한다.
2. 플레이어의 수를 입력받는다.
    - 플레이어의 수는 String으로 입력받는다.
    - null, empty, blank인지 확인한다.
    - 자연수인지 확인한다.
3. 각각의 유저로부터 수를 입력받는다.
    - 수는 String으로 입력받는다.
    - 수가 null, empty, blank인지 확인한다.
    - 유저의 수가 자연수인지 확인한다.
    - 잘못된 입력일 경우 Exception을 호출한 후 다시 입력받는다.
4. 컴퓨터가 생성한 난수와 유저가 입력한 수를 비교한다.
    - 컴퓨터가 생성한 수가 유저가 입력한 수보다 높을 경우 "더 높게! (남은 기회: _번)" 출력
    - 컴퓨터가 생성한 수가 유저가 입력한 수보다 낮을 경우 "더 낮게! (남은 기회: _번)" 출력
    - 맞았을 경우 몇 번만에 맞았는지 출력한다.
5. 정답을 맞힌 경우 "한판 더 하시겠습니까?"를 출력한 후 응답을 받는다
    - 사용자가 입력한 수가 empty, blank인지 확인
    - Y와 N을 IgnoreCase로 비교하여 입력문자 검증
6. 입력이 Y일 경우 다시 게임을 시작한다.


도메인
1. 컴퓨터: 숫자 생성기를 갖고있음 정답 숫자를 갖고있음
2. 플레이어: 플레이어의 이름을 갖고있음
3. Chance: 7번으로 초기화 플레이어의 기회
4. 숫자 생성기: 1~100 사이의 수를 생성
5. Output
6. Input
7. Game: 이 게임을 진행함 Player와 Chance를 Map으로 저장하여 사용. 게임의 총괄
